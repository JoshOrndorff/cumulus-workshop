# Cumulus 工作间

通过Cumulus连接Substrate区块链到波卡

在这个亲自动手的工作间，参与者们将开始一个类似波卡的中继链，注册平行链，实现跨链转账，以及通过使用Cumulus将他们自己的Substrate运行时逻辑转换成平行链。
