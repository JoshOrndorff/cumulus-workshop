# Substrate Cumulus 工作坊

通过 Cumulus 连接 Substrate 区块链到波卡。

在这个亲自动手的工作间，参与者们将开始一个类似波卡的中继链，注册平行链，实现跨链转账，以及通过使用 Cumulus 将他们自己的 Substrate 运行时逻辑转换成平行链。

## 版本

这个工作间成功测试运行在:

- Polkadot @ `46c826f`
- Parachain Template @ `465d539`
- Polkadot JS Apps @ `9f79c92`
